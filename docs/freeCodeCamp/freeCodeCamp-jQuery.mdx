## Introduction à jQuery

jQuery est l'une des nombreuses bibliothèques pour JavaScript. Il est conçu pour simplifier les scripts effectués côté client. La caractéristique la plus reconnaissable de jQuery est sa syntaxe de signe dollar (`$`). Avec lui, vous pouvez facilement manipuler des éléments, créer des animations et gérer les événements d'entrée.

-----





## 1. Découvrez comment fonctionnent les balises `script` et `document ready`

[Learn jQuery: Learn How Script Tags and Document Ready Work | freeCodeCamp.org](https://www.freecodecamp.org/learn/front-end-libraries/jquery/learn-how-script-tags-and-document-ready-work)

Nous sommes maintenant prêts à apprendre jQuery, l'outil JavaScript le plus populaire de tous les temps.<br />
Avant de pouvoir commencer à utiliser jQuery, nous devons ajouter des éléments à notre HTML.<br />
Tout d'abord, ajoutez un élément `script` en haut de votre page. Assurez-vous de le fermer sur la ligne suivante.<br />
Votre navigateur exécutera tout JavaScript à l'intérieur d'un élément `script`, y compris jQuery.

:::info instructions
Dans votre élément `script`, ajoutez ce code : `$(document).ready(function() {` à votre `script`. Puis fermez-le sur la ligne suivante (toujours à l'intérieur de votre élément `script`) avec : `});`.<br />
Nous en apprendrons plus sur les `functions` plus tard. La chose importante à savoir est que le code que vous insérez dans cette `function` s'exécutera dès que votre navigateur aura chargé votre page.<br />
Ceci est important car sans votre `document ready function`, votre code peut s'exécuter avant que votre HTML ne soit rendu, ce qui provoquerait des bogues.
:::

:::tip missions
- Vous devez créer un élément `script` en vous assurant qu'il est valide et qu'il comporte une balise de fermeture.
- Vous devez ajouter `$(document).ready(function() {` au début de votre élément `script`.
- Vous devriez fermer votre fonction `$(document).ready(function(){` avec `});`.
:::

## 2. Cibler des éléments HTML avec des sélecteurs à l'aide de jQuery

[Learn jQuery: Target HTML Elements with Selectors Using jQuery | freeCodeCamp.org](https://www.freecodecamp.org/learn/front-end-libraries/jquery/target-html-elements-with-selectors-using-jquery)

Nous avons maintenant un `document ready function`.<br />
Écrivons maintenant notre première instruction jQuery. Toutes les fonctions jQuery commencent par un `$`, généralement appelé opérateur de signe dollar ou bling.<br />
jQuery sélectionne souvent un élément HTML avec un sélecteur, puis fait quelque chose sur cet élément.

:::info instructions
Faisons rebondir tous vos éléments `button`. Ajoutez simplement ce code dans votre `document ready function` :

`$("button").addClass("animated bounce");`

Notez que nous avons déjà inclus la bibliothèque jQuery et la bibliothèque Animate.css en arrière-plan afin que vous puissiez les utiliser dans l'éditeur. Vous utilisez donc jQuery pour appliquer la classe `bounce` Animate.css à vos éléments `button`.
:::

:::tip missions
- Vous devez utiliser la fonction jQuery `addClass()` pour donner des classes `animated` et `bounce` sur vos éléments `button`.
- Vous ne devez utiliser jQuery que pour ajouter ces classes à l'élément.
- Votre code jQuery doit être dans la fonction `$(document).ready();`.
:::

:::note guide : Cibler des éléments HTML avec des sélecteurs à l'aide de jQuery

### Explication du problème
- Les sélecteurs jQuery vous permettent de sélectionner et de manipuler des éléments HTML.
- Ces sélecteurs commencent par le signe dollar et les parenthèses : `$()`.
- Vous pouvez « rechercher » (ou sélectionner) des éléments HTML en fonction de leur nom, identifiant, classes, types, attributs, valeurs d'attributs et bien plus encore.

#### Exemple :
```html
// Vous pouvez sélectionner tous les éléments <p> sur une page comme celle-ci =  $("p")
$(document).ready(function() {
  $("button").click(function() {
    $("p").hide();
  });
});
```

:::
