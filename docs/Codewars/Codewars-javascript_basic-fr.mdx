## 1. Créez votre première fonction JS `helloWorld`

[Training on Training JS #1: create your first JS function and print "Helloworld!" | Codewars](https://www.codewars.com/kata/571ec274b1c8d4a61c0000c8/train/javascript)

:::info instructions
En JavaScript, votre code s'exécute dans une fonction, complétons pas à pas votre première fonction JS.

Regardez cet exemple :

```js
  --------mot-clé "function"
  |       ----le nom de votre fonction  
  |       |    ---si nécessaire, certains arguments apparaîtront entre parenthèses
  |       |    |
function myfunc(){  ---------votre code de fonction commencera par "{"
  // vous devez taper votre code ici
}----------------------------se terminant par "}"
```

Si nous voulons en afficher à l'écran, nous pouvons peut-être utiliser `Document.write()` sur le Web, ou utiliser `alert()` pop votre message, mais Codewars ne prend pas en charge ces méthodes, nous devrions  utiliser `console.log()` dans votre fonction. Voir cet exemple :

```js
function printWordToScreen(){
  var somewords="This is an example."
  console.log(somewords)
}
```

Si nous exécutons cette fonction, `This is an example.` sera visible à  l'écran. Comme vous le voyez, `console.log()` est une méthode utile pour aider votre travail, vous deviendrez de plus en plus comme elle.

:::

- tapez le `console.log()` dans la ligne suivante (n'oubliez pas de mettre la chaîne entre crochets).

## 18. Les méthodes de l’objet String : `concat()`, `split()` et leur bon ami `join()`

[Train: Training JS #18: Methods of String object--concat() split() and its good friend join() | Codewars](https://www.codewars.com/kata/57280481e8118511f7000ffa/train/javascript)

Cette fois, nous apprenons deux méthodes pour diviser ou fusionner une chaîne de caractères : `split()` et `concat()`. Étudions également un bon ami de la méthode `split()` : `join()`. C'est une [méthode Array](https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Objets_globaux/Array). Leur utilisation :

```javascript
StringObject.indexOf(searchvalue,fromindex)
StringObject.lastIndexOf(searchvalue,fromindex)
StringObject.search(searchvalue/regexp)
```

`split()` peut diviser une chaîne en plusieurs parties par un séparateur spécifié. Le résultat est un tableau de chaîne de caractères. La chaîne fractionnée n'inclut pas le séparateur lui-même. L'une de ses utilisations  classiques est de diviser une phrase en un tableau de mots :

```js
var str="My name is John";
var words=str.split(" ");
console.log(words);
//output:
[ 'My', 'name', 'is', 'John' ]
```

Dans l'exemple ci-dessus, nous utilisons l'espace comme séparateur, divisant une phrase en 4 mots. Si nous spécifions les seconds paramètres, ce sera comme ceci :

```js
var str="My name is John";
var words1=str.split(" ",3);
console.log("words1:",words1);
var words2=str.split(" ",5);
console.log("words2:",words2);

//output:
words1:[ 'My', 'name', 'is' ]
words2:[ 'My', 'name', 'is', 'John' ]
```

Lorsque nous spécifions un nombre inférieur à 4, le résultat sera un nombre spécifié de chaînes ; si le numéro de la partition est trop grand, les résultats seront seulement les mêmes que les résultats par défaut.

Si nous utilisons la chaîne vide comme séparateur, nous obtiendrons un tableau de chaînes contenant chaque caractère :

```js
var str="My name is John";
var words=str.split("");
console.log(words);

//output:
[ 'M', 'y', ' ', 'n', 'a', 'm', 'e', ' ', 'i', 's', ' ', 'J', 'o', 'h', 'n' ]
```

Split peut utiliser des expressions régulières pour fractionner la chaîne. Mais parce que nous n'avons pas appris l'utilisation des expressions régulières, nous n'avons donc pas à apprendre cet usage.

`concat()` peut fusionner plusieurs chaînes en une chaîne comme celle-ci :

```js
var str="My".concat("name","is","John");
console.log(str);

//output:
MynameisJohn
```

En fait, nous voyons rarement l'utilisation réelle de `concat()`, car nous avons un moyen plus simple qui utilise l'opérateur + :

```js
var str="My"+"name"+"is"+"John";
console.log(str);

//output:
MynameisJohn
```

Mais même en utilisant l'opérateur +, les quatre mots ne sont pas la combinaison parfaite d'une phrase, car il n'y a pas de séparateur d'espace. Que devrions-nous faire ? Utiliser `join()` est le meilleur  choix.

`join()` est l'opération inverse de la méthode `split()`. Nous pouvons voir beaucoup de code dans l'utilisation réelle :

```js
var str="My name is John";
var words=str.split(" ");
console.log("use split():",words);
var s=words.join(" ");
console.log("use join():",s);
console.log("use split() and join():",str.split(" ").join(" "))
//output:
use split():[ 'My', 'name', 'is', 'John' ]
use join():My name is John
use split() and join():My name is John
```

Ok, la leçon est terminée, faisons une tâche.

**Tâche**
Implémentez une fonction qui accepte 2 arguments : `string` et `separator`.

L'algorithme attendu : divisez la chaîne en mots par des espaces, divisez chaque mot  en caractères séparés et rejoignez-les avec le `separator` spécifié,  joignez tous les 'mots' résultants en une phrase avec des espaces.

Par exemple :

```js
splitAndMerge("My name is John", " ")  ==  "M y n a m e i s J o h n"
splitAndMerge("My name is John", "-")  ==  "M-y n-a-m-e i-s J-o-h-n"
splitAndMerge("Hello World!", ".")     ==  "H.e.l.l.o W.o.r.l.d.!"
splitAndMerge("Hello World!", ",")     ==  "H,e,l,l,o W,o,r,l,d,!"
```

```js
function splitAndMerge(str, sp) {
  return str.split(" ").map(word => word.split("").join(sp)).join(" ");
}
///////////
function splitAndMerge(str,sp){
  var words = str.split(" ");
  var arr =[];
  var res = []
  for(i=0;i<words.length;i++){
   arr.push(words[i].split("").join(sp));
  }
  res = arr.join(" ");
  return res;
}
```
